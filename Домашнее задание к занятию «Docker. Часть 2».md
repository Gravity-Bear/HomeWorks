### Задание 1

**Напишите ответ в свободной форме, не больше одного абзаца текста.**

Установите Docker Compose и опишите, для чего он нужен и как может улучшить вашу жизнь.

#### Ответ

![задание1](https://github.com/Gravity-Bear/HomeWorks/assets/122636759/02f08bd8-4a93-42f6-9865-b21947102fcf)

Docker compose, с его помощью можно управлять несколькими контейнерами, ещё можно в одном файле собрать описания всех контейнеров, вместо того чтобы каждый из них описывать в отдельном файле. Всю конструкцию из множества контейнеров гораздо проще, удобнее и практичнее держать, развёртывать в одном файле.
